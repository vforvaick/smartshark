# Hybrid Wireshark Engine with Custom Investigation Workspace

Smartshark will use Wireshark tooling such as sharkd/tshark as the packet dissection and query engine, but will build a custom investigation workspace for evidence maps, evidence links, adjustable AI panels, and bounded agentic investigations. This keeps the product grounded in mature packet analysis capabilities without forcing the UX and domain model to inherit a webshark-style interface wholesale.
